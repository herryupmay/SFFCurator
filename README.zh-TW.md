# SFF 策展工具

[English version → README.md](./README.md)

每週論壇 SFF（科幻 / 奇幻 / 恐怖）主題策展助手。輸入一個主題，產出 5–8
部候選作品（書籍 / 電影 / 動畫 / 漫畫），整合多個來源的後設資料，並為
每部作品產生 200 字左右的繁體中文介紹草稿，方便人工潤飾後上線。

## 快速上手（Windows）

1. 從團隊共用資料夾**下載** `sff-curator.exe`。
2. **點兩下執行**。
   - Windows 可能跳出「來自不明的開發者」警告。請點 **更多資訊 →
     仍要執行**。檔案是開發者本人從原始碼編譯的，安全無虞。
   - 程式啟動後會留一個小型終端機視窗在背景，**不要關掉**，要等用完再關。
3. 瀏覽器會自動打開 `http://localhost:3000`。
4. **設定 API 金鑰**（首次使用一次即可）。點 ⚙ 設定，貼上：
   - **LLM 金鑰**（產生介紹用）。預設使用 Anthropic Claude，金鑰申請：
     <https://console.anthropic.com/>。也可以在同一個面板切換到 OpenAI、
     Google，或本機的 Ollama / llama-cpp-python 伺服器。
   - **TMDB 金鑰**（電影 / 影集查詢用，免費）：
     <https://www.themoviedb.org/settings/api>。**選填**，不填的話書籍
     /動畫 / 漫畫的查詢仍然正常，只是不會出現電影或影集的結果。
5. **輸入主題**，例如 `steampunk`、`蒸汽龐克`、`賽博龐克`、`cosmic horror`，
   按下「搜尋」。
6. 檢視結果，可以對個別作品點「生成繁中介紹」，也可以用上方工具列的
   「全部生成介紹」一次跑完。介紹文用人工潤飾後，按「匯出報告（.md）」
   下載 markdown 檔給論壇用。

## 你需要準備什麼

- Windows 10 或 11。
- 網路連線（搜尋階段與雲端 LLM 需要；本機 LLM 不需要）。
- 任何一個 LLM 服務商的免費或付費帳號（Anthropic / OpenAI / Google），
  *或者* 在你的電腦上跑本機 LLM（Ollama、llama-cpp-python、LM Studio）。
- 選填：免費 TMDB 帳號（電影 / 影集查詢用）。

## 常見問題（FAQ）

### 我的 API 金鑰安全嗎？

大致安全。誠實版本如下。

**金鑰存在哪裡：** 你瀏覽器的 localStorage，綁在 `localhost:3000` 這個來源
（origin）下。只有從這個來源載入的程式碼讀得到。本應用程式不會把金鑰寫入
硬碟，也不會送到 LLM 服務商以外的任何地方，更不會被打包進 .exe 裡。

**可以擋掉的威脅：**
- 同一個 Wi-Fi 上的其他裝置（伺服器只在 loopback 監聽）。
- 瀏覽器其他分頁的隨便網站（同源政策 + Host 標頭白名單）。
- 拿到 .exe 的其他朋友（每個人各自在自己的電腦上輸入自己的金鑰）。

**擋不掉的威脅（任何瀏覽器型 BYOK 工具都一樣，例如 Slack、Discord、
GitHub Desktop）：**
- 你電腦上的惡意程式。
- 別人趁你電腦沒鎖時打開瀏覽器開發者工具（DevTools）讀 localStorage。
- 有 localhost 權限的惡意瀏覽器擴充功能。

**實務建議：** 為這個工具**單獨申請**一把 API 金鑰，不要跟你日常使用的
共用。再到該服務商的計費頁面**設一個月度預算上限**（5 美元就綽綽有餘）。
這樣即使最壞情況金鑰外洩，損失也有上限，而且只需要重置這把金鑰，不會
影響你其他用途。Anthropic、OpenAI、Google 都支援單把金鑰的支出上限。

### 用 API 會在我原本的 Claude/ChatGPT/Gemini 訂閱費之外另外收費嗎？

會。所有主要服務商的「訂閱版」和「API」都是分開計費的兩個產品：

- **Anthropic** — Claude.ai Pro（每月 20 美元）**不含** API 存取。你需要
  另外開 console.anthropic.com 帳號，預付額度或設定計費。
- **OpenAI** — ChatGPT Plus / Pro / Team 只能在 chatgpt.com 用。API 要另
  開 platform.openai.com 帳號設定計費。
- **Google** — Gemini App 訂閱與 API 分開。到 ai.google.dev 申請 API 金鑰
  時會送你**免費額度**，一般輕度使用大概就夠用。
- **本機 LLM**（Ollama、llama-cpp-python、LM Studio）— 完全免費，靠你
  電腦的 CPU/GPU 跑。

**單次策展（8 部作品）的概略費用：**

| 服務商 | 單次 | 每月（每週一次，共 4 次） |
|---|---|---|
| Claude Sonnet 4.5 | 約 0.05–0.07 美元 | 約 0.20–0.30 美元 |
| GPT-4o | 約 0.03–0.05 美元 | 約 0.15–0.20 美元 |
| Gemini 1.5 Pro | 差不多；免費額度大概夠用 | 大概 0 美元 |
| 本機 Gemma | 0 元 | 0 元 |

每週一次的論壇節奏，整體支出落在「比一杯咖啡便宜」的範圍。

### 我該選哪一家？

**繁中品質最好（開箱即用）：** Anthropic Claude。台灣用語最乾淨，黃色
警示最少。本工具的提示詞就是針對它調的。

**想免費試用：** Google Gemini 的 API 免費額度，或者用 llama-cpp-python
跑本機 Gemma。

**追求隱私 / 完全離線：** 用本機 LLM。設定步驟在 BUILD 文件裡。

### 如果某個來源都沒有結果怎麼辦？

博客來 / Readmoo / ISFDB 的爬蟲是直接解析這些網站的 HTML，這些網站偶爾會
改版。如果某個來源持續 0 結果而其他來源正常，就是選擇器（selector）需要
更新了。這部分需要開發者處理，請回報給開發者，附上你搜尋的主題。

### 我的資料存在哪裡？

- **API 金鑰：** 只存在你這台電腦的瀏覽器 localStorage 裡。
- **搜尋結果：** 只存在瀏覽器記憶體中，關閉分頁就沒了。
- **生成的介紹：** 只存在瀏覽器記憶體中。按「匯出報告」會把整份報告
  下載到你的「下載」資料夾，存成 .md 檔。
- **沒有任何伺服器端的儲存。** 整個程式只在你自己電腦上跑，沒有外部
  伺服器。

### 我要怎麼關掉程式？

先關閉瀏覽器分頁，再關閉啟動程式時跳出的那個終端機視窗（只關瀏覽器
不會結束程式，背景會繼續跑——不影響使用，只是會佔一點點記憶體，到你
關掉終端機或重開機才會結束）。

## 出問題時

- **瀏覽器沒自動打開？** 自己打開瀏覽器，輸入 `http://localhost:3000`。
- **跳出「Port 3000 已被使用」？** 表示有別的程式佔用了這個埠號。把那個
  程式關掉，或者重開機後再試。
- **某個來源都沒有結果？** 參考上方常見問題。其他來源仍然會正常運作。
- **繁中介紹混入大陸用語？** 用非 Claude 模型有時會這樣。程式會把可疑
  字詞用黃色標記，可以手動改掉。狀況太糟的話，可以到設定切回 Claude。
- **其他問題？** 請告訴開發者，附上你搜尋的主題、所選的服務商與模型，
  以及終端機視窗顯示的訊息。

## 隱私

整個程式只在你電腦上跑，沒有外部伺服器。程式直接與 API 服務商
（Anthropic、TMDB 等）溝通。你的金鑰、搜尋紀錄、產生的報告都不會離開
你的電腦，唯一例外只有實際的 API 呼叫本身。
